# Cách chạy
Áp dụng Class/ID, HTML semantic; tổ chức thư mục /assets/img/css. (Điểm cộng) JS thuần cho hiệu ứng cơ bản: menu mobile, modal, carousel.
Về html semantic, nhóm sử dụng các thẻ như <img>, <header>, <footer>.
Về tổ chức thư mục, nhóm xây dựng file CSS External để đảm bảo tính linh hoạt khi thiết kế.
Về class và id, nhóm sử dụng những class như “header_search” hay “slider” trong khi code để dễ dàng trình bày trên CSS cũng như xây dựng nội dung.
Deploy lên GitHub Pages (public),.
Nhóm thực hiện theo hướng dẫn của giáo viên theo canvas, bắt đầu từ việc tạo tài khoản GitHub, liên kết website và deploy website lên GitHub.
